# Artifacts

Things the crew made. Creative output.

## What belongs here

- Finished or in-progress works
- The Prism (explorable memory space)
- Cup of the Sea (the crew's first song)
- Tools we built, even deprecated ones

## What doesn't belong here

- Experiments → `explorations/`
- Internal reflections → `echoes/`
- Pending decisions → `proposals/`
- Technical infrastructure → `infrastructure/`
